Разработка мобильных приложений

Основная задача, которую будут решать студенты с использованием программного обеспечения данной мастерской – создание мобильных приложений, сочетающих в себе такие обязательные качества, как безотказная работа на одной из мобильных операционных платформ (Apple iOS, Google Android, Windows Phone), удобный пользовательский интерфейс, и многофункциональность.

Оснащение мастерской используется для развития следующих компетенций:

  • Использования технологий для разработки мобильных приложений;
  • Использования технологий для работы с базами данных;
  • Использования технологий для работы с различными протоколами обмена данными;
  • Строительство приложения со сложной логикой переходов;
  • Работы со стандартными сервисами платформы (например: google services, apple);
  • работы со встроенными устройствами для получения данных (гироскоп, GPS, акселерометр);
  • планирования тестирования (например, тестирование элементов, объемное тестирование, комплексное тестирование, приемочное тестирование);
  • отладки мобильного приложения и устранения ошибок;
  • отчёта о процессе тестирования мобильных приложений.

ИТ-решения для бизнеса на платформе "1С: Предприятие 8"

В мастерской установлена информационно-техническая платформа «1С: Предприятие 8», которая позволяет разрабатывать на её основе различные прикладные решения для бизнеса, включая решения ERP-класса.

Оснащение мастерской используется для развития следующих компетенций:

  • анализа и проектирования программных решений;
  • использования последней версии среды и средств разработки технологической платформы 1С:Предприятия 8»;
  • определения и интегрирования требуемых библиотек в программное решение;
  • настраивания регламентных процедур в СУБД, используемых для хранения данных 1С:Предприятия (Microsoft SQL Server, PostgreSQL, IBM DB2 или Oracle Database);
  • строительства многоуровневых приложений;
  • создания прикладных решений;
  • составления плана тестирования (например, модульное тестирование, объемное тестирование, интеграционное тестирование и приемочные испытания);
  • разработки тест-кейсов и проверки их результатов; устранения и исправления ошибок; составления отчетов о процессе тестирования.

Разработка компьютерных игр и мультимедийных приложений

В мастерской студенты осваивают компетенцию создания компьютерных программ высокой сложности, предназначенных для обучения и развлечения пользователей. Проектирование игр, как программных систем, требует умений анализа требований к программной системе, навыков программирования, особенностей технических средств и сред разработки. Кроме того, реализация компьютерных игр требует знаний в области разработки алгоритмов, проектирования и разработки интерактивных приложений, двумерной и трехмерной компьютерной графики, кроссплатформенном программировании и т.д.

Оснащение мастерской используется для развития следующих компетенций:

  • работы с техническим описанием игры;
  • использования имеющихся инструментов и возможностей платформе Unity;
  • проектирования и строительства пользовательских интерфейсов для игровых продуктов;
  • программирования игровой механики и реализации игрового процесса согласно технического описания игры;
  • оформления кода и использования комментирование согласно общепринятым нормам;
  • выстраивания масштабируемой архитектуры проекта;
  • проведения тестирования и отладке программных решений;
  • разработки функционирующего программного продукта для конечного пользователя.

Веб-дизайн и разработка

Уникальность мастерской заключается в возможности подготовки специалистов, способных разрабатывать веб-приложения разной степени сложности, от персонального сайта до портала предприятия. Для этой цели студенты осваивают компетенции как в области веб-технологий, так и в сфере графического дизайна.

Оснащение мастерской используется для развития следующих компетенций:

  • создания, использования и оптимизирования изображения для веб-сайтов;
  • выбора дизайнерского решение, которое будет наиболее подходящим для целевого рынка;
  • программирования на стороне клиента;
  • программирование на стороне сервера;
  • установки, настройки и обновления системы управления контентом;
  • установки, настройки и обновления плагинов/модулей CMS;
  • создания пользовательских тем/шаблонов для системы управления контентом.

Программные решения для бизнеса

В мастерской отрабатываются навыки решения задач в области индивидуальной разработки или персонализации программных решений, управления системами. Студенты, с использованием установленного программного обеспечения, осуществляют бизнес-аналитику для предоставления решений, упрощающих и автоматизирующих рутинные офисные и бизнес-процессы; овладевают компетенциями по обучению пользователей применению прикладных программ.

Оснащение мастерской используется для развития следующих компетенций:

  • анализа и проектирования программных решений;
  • использования системы управления базами данных для построения, хранения и управления данными для требуемой системы (MySQL или MS SQL Server);
  • изменения существующих кодов и написания нового кода "клиент-сервер" на базе программного обеспечения (.NET или Java);
  • определения и интегрирования соответствующих библиотек и Фреймворков в программное решение;
  • строительства многоуровневые приложения;
  • разработки мобильного интерфейса для клиента на основе серверной системы;
  • составления плана тестирования; разработки тест-кейсов и проверки результатов тест- кейсов;
  • устранения и исправления ошибок; составления отчета о процессе тестирования;
  • разработки однопользовательских и высоконагруженных программных решений.
